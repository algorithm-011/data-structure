- First In, First Out

## 종류

- 선형(Linear): 말그대로 막대 모양으로 이뤄진 구조. 메모리의 빈 공간을 사용하려면 모든 자료를 꺼내거나 옮겨야함.
- 환형(Circular): 선형 큐의 문제점(메모리 공간이 남았음에도 불구하고 활용하지 못하는 현상)을 보완한 형태.
