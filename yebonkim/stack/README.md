# Stack 종류

스택은 기본적으로 Descending Stack 이다

낮은 주소는 주소 숫자가 작은 주소 <br>
높은 주소는 주소 숫자가 높은 주소


1. Ascending Stack
- 데이터가 낮은 주소에서 높은 주소로 쌓이는 스택

2. Descending Stack
- 데이터가 높은 주소에서 낮은 주소로 쌓이는 스택

3. Full Stack
- 포인터가 방금 실행한 push pop의 데이터를 가리키는 스택

4. Empty Stack
- 포인터가 방금 실행한 push pop 다음 주소를 가리키는 스택
